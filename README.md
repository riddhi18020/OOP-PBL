This repository contains practical implementations of problem-solving exercises for the Object-Oriented Programming subject.
# Object-Oriented Programming (OOP) Practicals

## Subject Details
- **Subject:** Object-Oriented Programming (BE04000231)  
- **Year/Semester:** 2nd Year (4th Semester)  
- **Academic Year:** 2026  

---

##  Practical Questions

1. Convert distance from meters to feet and display the result up to two decimal places.

2. Solve a system of two linear equations using Cramer's Rule.

3. Check whether a given character is a vowel or a consonant.

4. Calculate Body Mass Index (BMI) using given weight and height.

5. Calculate the area of a triangle using Heron’s formula (with validation).

6. Create a `Rectangle` class with methods to calculate area and perimeter.

7. Create an `Employee` class to input and display employee details.

8. Create a `Point` class with default, parameterized, and copy constructors.

9. Compare two rectangles based on their area.

10. Design a `BankAccount` class with a static interest rate.

11. Implement a college admission system using an inner class.

12. Demonstrate method overloading for volume calculation.

13. Create a `Shape` class and derive `Triangle` and `Rectangle` classes.

14. Implement a banking system using inheritance.

15. Demonstrate method overriding using `Employee` and `Manager`.

16. Implement an interface to classify division based on average.

17. Implement multiple interfaces for exam result processing.

18. Create an abstract class `Vehicle` with subclasses `Car` and `Bike`.

19. Use packages to generate a student mark sheet.

20. Handle exceptions using command-line arguments.

21. Create a voting eligibility checker with exception handling.

22. Implement a custom exception for book availability.

23. Create threads to print numbers at different intervals.

24. Print numbers sequentially using multiple threads.

25. Print multiplication tables using synchronized methods.

26. Count characters, words, and lines in a file.

27. Write and read student records from a file.

28. Analyze file data using FileReader/BufferedReader.

29. Use `ArrayList` to store and analyze marks.

30. Count word frequency using `HashMap`.

31. Simulate a music playlist using `LinkedList`.

32. Implement a generic search method using `LinkedList`.

33. Display styled text using JavaFX.

34. Create a registration form with validation.

35. Display grade distribution using a bar chart.
